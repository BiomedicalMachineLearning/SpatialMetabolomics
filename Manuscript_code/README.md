# SpatialMetabolomics

The codes can be used to reproduce Figures in the manuscript entitled: "Spatial pharmaco-multiomics reveals drug distribution, metabolic niches, and spatially constrained resistance in medulloblastoma". 

Codes for Figures 1 and 2 can be found in the Visium_Analysis (Fig 2c) and Xenium_Analysis (Fig 2d, e, f).
codes for Figue 3 can be found in the Joint_ST_SM_Analysis (Fig 3a-f in General_analysis_Andrew.ipynb). Codes for Fig 3g, h are at Visium_Analysis 

Codes for Figures 4, 5, 6, and 7 are in Joint_ST_SM_Analysis.
