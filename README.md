# SpatialMetabolomics

The Manuscript_code folder contains scripts used to generate figures in the manuscript entitled: "Spatial pharmaco-multiomics reveals drug distribution, metabolic niches, and spatially constrained resistance in medulloblastoma". 

The authosr include: Tuan Vo, Cedric S. Cui, Aitor Benedicto, Yibi Chen, Andrew Causer, Xiao Tan, Eun Ju Kim, Hani Vu, Albert Xiong, Mark Hodson, Yak Deng, Ana Maria Romero Jimenez, Ning Liu, Brett Hamilton, Thomas Robertson, Laura Genovesi, Brandon Wainwright, Jenny Fung, John Lee, Quan Nguyen.

For questions and suggestions, please contact quan.nguyen@qimrb.edu.au
